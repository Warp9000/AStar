# AStar

## Controls

| Key   | Function            |
| ----- | ------------------- |
| LMB   | Paint unwalkable    |
| RMB   | Paint Walkable      |
| C     | Clear grid          |
| P     | Set Player Position |
| T     | Set Target Position |
| M     | Generate a maze     |
| Space | Find AStar path     |
| S     | Find smothed path   |
| K     | Find corner path    |
